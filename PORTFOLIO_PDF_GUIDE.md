### 제거할 내용

#### 1. 긴 코드 블록
**README**:
```swift
// 전체 ExReactorKit.swift 코드 (30줄)
extension Reactor {
    func run(
        operation: @Sendable @escaping (SendFunction) async throws -> Void,
        onError: @escaping (Error) -> Mutation = { _ in fatalError() }
    ) -> Observable<Mutation> {
        return Observable.create { observer in
            let task = Task { @MainActor in
                ...
            }
            return Disposables.create {
                task.cancel()
            }
        }
    }
}
```

**PDF**:
```
Task → Observable 변환
• Task 생성하여 async 실행
• Observable.create로 Mutation 스트림화
• Task 취소 자동 해제
```

#### 2. 상세한 설명
**README**: "async 작업이 완료될 때마다 `send()`로 Mutation을 전달하면, RxSwift Observable이 이를 구독자에게 순차적으로 배출합니다..."

**PDF**: "send() → Observable → State"

#### 3. 파일 경로
**README**: `Core/Extension/ExReactorKit.swift`

**PDF**: 제거 (필요 없음)
